== Description ==

A very simple plugin to change Date, Time, Months and Days in Bangla. It will change default Archive and Calendar Widget too.
It's a Beta version, It may have some issue or conflict. Please let me know by support forum, so, I can fix these and make it rich enough.

Please rate this plugin if you think it is helpful.


== Installation ==

1. Upload \`plugin-name.php\` to the \`/wp-content/plugins/\` directory
2. That's it. You are done. Now check, how it works

== Frequently Asked Questions ==

= Will it change my default Calendar? =

Yes. It will change default Calendar and Archive widget.
